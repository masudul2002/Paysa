# Paysa Documentation

Paysa is an offline-first personal finance manager designed with future cloud synchronization. This documentation pack is the source of truth for product, architecture, engineering, QA, release, and AI-assisted development decisions.

Start with `00_Documentation_Index.md`, then follow the reading order in `00_Documentation_Maintenance_Strategy.md`.

Rules:
- Documentation changes precede implementation changes.
- Architecture decisions are recorded before code is written.
- Future cloud sync assumptions must be labeled separately from MVP commitments.
- This pack contains planning documentation only; it does not contain Flutter, Dart, widget, or database implementation code.
